# finance-ledger
finance-ledger is a beginner-friendly Spring Boot project that models a small banking ledger using double‑entry accounting. It includes JWT authentication, PostgreSQL with Flyway migrations, Testcontainers for integration tests, and OpenAPI/Swagger docs—perfect to practice real-world REST API design, validation, transactions, and idempotency.

## Getting Started

This repository now includes a minimal Spring Boot setup with an in-memory H2 database. After building the project you can run `LedgerApplication` and visit `/accounts` to see the sample data loaded from `data.sql`.
